# tty-chance-games
Simple terminal chance games: includes 
> coinflip
> russian roulette
# Installation

## Linux systems
1. Run `chmod +x install.sh`
2. type `./install.sh`
The install script should copy the file over to /usr/bin. From there you can run `tty-chance-games`
